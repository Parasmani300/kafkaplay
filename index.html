Kafka Play

A versatile tool for producing and consuming messages in Kafka

Key Features

Message Production

Easily produce messages for testing purposes or publishing records to higher environments.

Message Consumption

Consume messages from Kafka topics to validate data or perform troubleshooting.

Kafka Administration

Perform administrative tasks like creating topics, resetting offsets, and describing groups and topics.
